linux useful command

The file contain the usual shell command that I used in my works.

1 directory and file manipulate

a). mv moving_file  new_file  #move or rename a file
b). mkdir/rmdir dir_name     #make a dir or delete a dir
c). rm -rf XXX      # delete all file or directory contain directories or files

2 zip/unzip

a) tar czvf zip_file dir # zip the dir into a zip file
b) tar xzvf zip_file  desdir  # unzip a zip file to a dir
c) rar a zip_file sourcedir
d) rar x zip_file destdir

3 screen

screen -S yourname
screen -D yourname
screen -r yourname
screen -wipe
screen -ls

4 scp

a) scp -r sourcedir destination_dir(username@IP:pwd/dir)

5 user

su user  #from root to user
sudo su  #from user to root
sudo password root #set passowrd
~ represent home dirctory of a current user
adduser /passwd
gpasswd -a/-d userid groupid
uname -a  # system imformation
cd ~
vim .bashrc  # change a user profile using the system
PATH=/usr/local/MATLAB/R2013a/bin: $PATH
export PATH

6 ls

ls -a 
ls -sh
ls -l

7 free

df
du 
free

8 chmod

chmod 777 file
ps -ef|grep  |cut -c 15-20|xargs kill -9
ps -ef|grep XX |awk '{print $1,$2}'

9 install , uninstall and download

sudo apt-get remove XXX  #remove the application without disk files
sudo apt-get purge XXX #purge disk files clearly
sudo apt-get install XXX  # This command conducts downloading and installing the software from the repository that was maintented by association.
wget url   # download the data from the given url
